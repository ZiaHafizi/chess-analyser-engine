# ♟️ Chess Analyzer (Single File Edition)

A lightweight **browser-based chess analysis tool** built entirely in a single HTML file.
No installations. No frameworks. No backend. Just open the file and start analyzing chess positions instantly.

---

# ✨ About The Project

Chess Analyzer is a minimal yet powerful chess analysis tool designed for players who want fast position evaluation without installing heavy chess engines or applications.

Unlike traditional chess software that requires complex setup, this project runs **entirely in the browser** using only HTML, CSS, and JavaScript inside a single file.

This makes the project extremely portable, easy to share, and simple to modify.

The philosophy of this project is:

* Simplicity
* Accessibility
* Speed
* Lightweight architecture

Anyone can download the file and start analyzing chess positions immediately.

---

# 🚀 Key Features

### ♟️ Interactive Chess Board

A fully functional chess board allowing users to explore positions and test moves directly in the browser.

### 🔍 Position Analysis

Analyze chess positions quickly and see evaluation changes as moves are played.

### ⚡ Zero Installation

No Node.js
No Python
No engines to install

Just open the file in your browser.

### 📦 Single File Architecture

The entire project is contained in **one HTML file (~45KB)**.

Benefits include:

* ultra portable
* easy hosting
* instant loading
* no dependencies

### 🌐 Runs Everywhere

Works on:

* Chrome

* Safari
* Mobile browsers
